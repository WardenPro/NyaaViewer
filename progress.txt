## Key Architecture Decisions
- HashRouter (not BrowserRouter) for Electron file:// protocol compatibility
- All nyaa.si requests go through main process (no CORS)
- mpv spawned as child process with Unix socket IPC
- Storage uses app.getPath('userData') for cross-platform consistency
- Zustand store in renderer, main process keeps own state (AD key, mpv)
- fast-xml-parser for RSS parsing (lighter than xml2js)
